useDeleteStoreValue(key) → {function}

Returns a function to delete a variable in the store. You want to use this hook when you just need to delete a value in the store, not read or set a value from it.

Parameters:
Name Type Description
key string

the name of the variable to set in the store
Source:
  • useStore.js, line 78
Returns:
  • a function to delete a variable in the store with the given name. When used, this function return a promise that resolve nothing, thus you can use deleteValue('a value').then(() => {doSomething() //when the store did update})
Type
function
Example
import {useDeleteStoreValue} from 'react-context-hook'
const deleteUsername = useDeleteStoreValue('username')
<button onClick={()=> deleteUsername('my_username')}>set username</button>
